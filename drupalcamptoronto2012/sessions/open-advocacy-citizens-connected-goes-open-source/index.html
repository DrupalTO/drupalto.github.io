Open Advocacy - Citizens Connected goes open source

Tweet

Speaker(s)

Sebastian Lesch

Citizens Connected is a tool that was created 2 years ago to assist organizations in online advocacy. The David Suzuki Foundation has used this tool to create dozens of campaigns and send thousands of targeted emails throughout the years.

Citizens Connected was originally written in Drupal 6, but has since been re-written in Drupal 7. The new D7 version takes advantage of entities, fields, and other D7 goodies, but the potential of the project is too large to continue to be maintained by the small group of existing developers. There is already a solid use case for this module and existing momentum, but there is huge potential to take it even further. We're happy to announce that we will be making this tool open source and want your help.

The tool uses ctools so developers can create "look up" plugins so supporters of a campaign can easily find their specific representative and send a message to them. Currently we've been integrating with Open North for this functionality, and they've done a great job of making this data easily accessible. Data integration plugins have been written into the system to integrate it with Mail Chimp, Emma, and SalesForce, although those plugins need to be further abstracted. The tool also has a few small bugs and features that need to be built out. But, Citizens Connected is well on its way to being a campaign tool that's fully integrated with Drupal.

Organizations are poised and excited to be able to use Citizens Connected and make their voices heard. Come and find out how you can help out on this project.
